# VRRP (Virtual Router Redundancy Protocol)

This folder contains CCIE-style labs and notes for **VRRP**, an open-standard First Hop Redundancy Protocol (FHRP).

## Files
- **VRRP_Lab.md** – Configure and verify VRRP master/backup gateway redundancy with internet reachability tests.
- **images/** – Topology screenshots used by the labs.
